# 这是一个python脚本，输入天天基金网某个基金的网址，自动抓取抓取基金历史净值数据。
# 通过这个脚本，你可以抓取任何一个在天天基金网站上的基金的历史净值数据。例如
# [易方达上证50增强A (110003)](https://fundf10.eastmoney.com/jjjz_110003.html)
# [易方达蓝筹精选混合 (005827)](https://fundf10.eastmoney.com/jjjz_005827.html)
